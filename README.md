# PIC18F4580 Timer1 Delay Generation

## 💡 Overview

This project demonstrates how to use the **Timer1 peripheral of the PIC18F4580 microcontroller** to generate accurate time delays. Timer1 is configured as a 16-bit timer with a prescaler of 1:8 and a preload value that produces an approximate delay of **2.5 ms** per overflow.

The generated delay is used to toggle **PORTD** between HIGH and LOW states, creating a square wave output that can be observed using an oscilloscope or LEDs.

This project introduces:

* Timer1 Configuration
* Delay Generation Using Hardware Timers
* Timer Overflow Polling
* Digital Output Toggling
* Square Wave Generation
* Embedded C Programming with PIC18F4580

## ⏰ What is Timer1?

Timer1 is a **16-bit hardware timer/counter** available in the PIC18F4580 microcontroller.

Unlike software delay loops, Timer1 uses dedicated hardware to count clock pulses, providing more accurate timing.

Timer1 can be used for:

* Delay Generation
* Event Counting
* Frequency Measurement
* Real-Time Clock Applications
* Interrupt-Based Timing

---

## 🧮 Delay Calculation

The delay generated by Timer1 is calculated using:

```text
Delay = (65536 − Timer Preload) × Tosc × Prescaler
```

Where:

| Parameter     | Value  |
| ------------- | ------ |
| Fosc          | 20 MHz |
| Tosc          | 50 ns  |
| Prescaler     | 8      |
| Timer Preload | 0xE796 |

### Preload Conversion

```text
0xE796 = 59286
```

### Calculation

```text
Delay = (65536 − 59286) × 50 ns × 8
```

```text
Delay = 6249 × 50 ns × 8
```

```text
Delay = 6249 × 400 ns
```

```text
Delay ≈ 2.5 ms
```

### Practical Observation

In simulation, the timer delay is repeatedly used to generate a waveform with an approximate total period of:

```text
10 ms HIGH
10 ms LOW
```

Resulting in:

```text
20 ms total waveform period
```

## ⚙️ How It Works

### Step 1

Configure PORTD as output.

### Step 2

Load Timer1 preload value:

```text
TMR1H = 0xE7
TMR1L = 0x96
```

### Step 3

Enable Timer1:

```text
T1CON = 0x79
```

### Step 4

Set PORTD HIGH.

### Step 5

Wait until Timer1 overflows.

### Step 6

Clear overflow flag.

### Step 7

Reload Timer1.

### Step 8

Set PORTD LOW.

### Step 9

Repeat continuously.

## 💻 Complete Source Code

```c
#include <pic18.h>

void timer();

void main(void)
{
    TRISD = 0x00;    // Configure PORTD as output

    TMR1L = 0x96;    // Load Timer1 Low Byte
    TMR1H = 0xE7;    // Load Timer1 High Byte

    T1CON = 0x79;    // Enable Timer1 with Prescaler 1:8

    while (1)
    {
        LATD = 0xFF; // PORTD HIGH
        timer();

        LATD = 0x00; // PORTD LOW
        timer();
    }
}

void timer()
{
    while (TMR1IF == 0); // Wait for Timer1 Overflow

    TMR1IF = 0;          // Clear Overflow Flag

    TMR1L = 0x96;        // Reload Low Byte
    TMR1H = 0xE7;        // Reload High Byte
}
```
